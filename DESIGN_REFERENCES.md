# Design References - Ayah & Surah Apps

## Key Design Elements Observed:

### Color Schemes:
1. **Ayah App (Image 1)**: 
   - Cream/Beige background
   - Dark brown text
   - Subtle gold accents
   - Minimalist and elegant

2. **Surah App (Images 2-4)**:
   - Purple gradient background
   - White cards
   - Modern and vibrant
   - Clean typography

3. **Naqli App (Image 5)**:
   - Teal/Turquoise primary color
   - White and light backgrounds
   - Modern Islamic design
   - Professional layout

4. **Modern Quran Apps (Image 6)**:
   - Dark blue/teal backgrounds
   - Gold/yellow accents
   - Clean card-based layout
   - Professional typography

### Layout Patterns:
- **Sidebar Navigation**: List of surahs on the left
- **Main Content Area**: Verse display in the center
- **Bottom Sheet/Modal**: For tafsir details
- **Tab Navigation**: For different tafsir sources
- **Search Feature**: Prominent search bar
- **Bookmarks/Favorites**: Heart icon for saving

### Typography:
- Large, readable Arabic font (Amiri or similar)
- Clear hierarchy between verse and tafsir
- Proper line spacing for readability
- RTL layout support

### Interactive Elements:
- Smooth transitions between surahs
- Expandable/collapsible sections
- Swipe navigation
- Copy to clipboard functionality
- Share verse feature

## Recommended Design Direction:
**Combine the elegance of Ayah with the modernity of Surah**
- Use a sophisticated color palette (dark teal or dark green with gold)
- Implement clean card-based layout
- Include search and bookmark features
- Use professional typography
- Maintain RTL layout throughout
